# Notes App

Notes App is a simple web application built with Node.js and Express.  
The purpose of this project is to create a basic notes application where users
will be able to create, edit, and manage their personal notes in one place.

At this stage, the project focuses on setting up the server, organizing the
project structure, and creating a landing page for future development.
